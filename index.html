Projects

The works of passion that I find myself doing to distract myself from the daunting reality of adulting, only to realize they've become new obsessions in their own right.

Momentag

In Momentag your goal is to grapple and dash around to reach high speeds while competing against other players in various competitions such as keepaway and infection tag. This project is made with Unity and C#. My role within this project was Lead Developer.

Park builder

In Park builder your goal is to make an ascetic and pleasing park. This project is made with Unity and C#. The models are made using MagicaVoxel.

Custom Terrain System for Unity

The goal of this system is to create a terrain system that allows for a very large world. The  system supports a maximum of 128km x 128km. The terrain has native support for cave cliffs and  overhangs. The project is made with Unity and C#.
